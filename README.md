# 🍔 Yummy Bite - Landing Page

## 🚀 Overview
A modern and responsive restaurant landing page built from a Figma design.  
The project focuses on clean UI, smooth animations, and interactive user experience using pure JavaScript.

---

## ✨ Features
- 📱 Fully Responsive Design
- 🔝 Scroll To Top Button
- 📂 Interactive Navbar (Open / Close Menu)
- 🌙 Dark & Light Mode (saved with LocalStorage)
- 🎬 Smooth Scroll Animations (IntersectionObserver)
- 🎯 Hero Section Animation
- 💬 Client Testimonials Slider
- ⚡ Smooth User Experience

---

## 🛠️ Tech Stack
- HTML
- CSS
- SASS
- JavaScript (Vanilla)
